# Hybrid-Deep-Learning-and-Bayesian-Network-for-Real-Time-Book-Recommendation
A hybrid real-time book recommendation system combining Deep Learning and Bayesian Networks. Uses DistilRoBERTa for sentiment, MiniLM embeddings with FAISS for semantic retrieval, and a BN for explainable re-ranking. Achieves improved Precision@5 with personalized and interpretable suggestions.
